IOTA CLUB

IOTA is the Project Club of IIIT Sri City. The aim of this club is to allow people to collaborate and make real world projects and to pass on the knowledge to the new comers. We have conducted and attended a lot of session over a wide range of topics, and we attempt to do the same for UG-1. Below are the details of the planned sessions, and projects made by the last year UG-1 batch.

Planned Sessions

Web Developement 1: HTML

The Sessions will contain an introdutory session about HTML and its uses, how it works, the DOM structure. All the valid tags and attributes will be taught and the session will be hands-on with DIY tasks as well.

Web Developement 2: CSS

This session will focus towads the use of CSS(Cascading Style Sheets) in Web Developement, the diffrent types of styling properties which are available and how they affect the look of HTML document.

Web Developement 3: JS

JavaScript is most popular scripting language for Web Developement, it enables us to allow the HTML document to be interactive. It allows the users to use website not just as a documents but an actual application.

Web Developement Tools: Firebase

Firebase is a mobile and web application development platform currently managed by Google inc., it is one of the best options to easily integrate authentication/login, database system, analytics, etc. for your Web Application.

Git + Github

Git is the most popular VCS(Version Control System) which is used by developers around the world to easily manage projects. It allows you to track your changes/modifications in your projects, have many people work on same project easily, rollback to a previous version and many more.

Containerized Applications: Docker

Docker enables us to run applications/programs in isolated containers. Containers in simple words are like Virtual Machines but instead of running an entire OS, run a single application allowing them to be really lightweight and easier to manage.
